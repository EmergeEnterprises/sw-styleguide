This styleguide is created with these values in mind:

Clarity

The styleguide is intended to be the source of truth for the latest styles.

Maintainability

It should be easy to maintain and update for both design and development.

Speed

Design and development should be able to move faster using a styleguide first mindset.

Content

Page titles, navigation items, table labels and buttons should appear title case: "Your Store" or "Save Changes"
Text input fields should appear sentence case: "Email address" or "Confirm new password"
